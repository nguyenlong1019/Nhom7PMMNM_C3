# các chức năng làm sạch dữ liệu 
loại bỏ các cột null hoặc có giá trị bằng 0
nếu trong cột có số lượng giá trị null hoặc 0 lơn hơn hoặc bằng 80% thì xóa cột đó đi
nếu trong cột có số lượng giá trị null < 80%
    < 50%: lấy giá trị trung bình numpy.mean()
    < 65%: lấy giá trị ở giữa numpy.median()
    < 80%: lấy giá trị numpy.mode(): giá trị xuất hiện nhiều nhất
=> kết quả sau khi làm sạch thì sẽ là giữ liệu được điền đầy đủ
